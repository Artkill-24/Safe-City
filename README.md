# Safe City

A decentralized application (DApp) for improving urban security through blockchain technology.

## Features
- Blockchain-verified security reports
- SCIT token rewards
- Reputation system
- Community validation
- AI-powered analysis

## Tech Stack
- Solidity/Arbitrum
- React/Next.js
- Tailwind CSS
- Llama AI
